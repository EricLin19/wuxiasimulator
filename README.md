# 小小侠客网页版原型

## 运行方式

直接打开 `index.html` 即可游玩。

如浏览器限制本地脚本，可在此目录启动静态服务器：

```powershell
python -m http.server 8877 --bind 127.0.0.1
```

然后访问：

```text
http://127.0.0.1:8877
```

## 已实现内容

- 三名角色与六件开局宝物
- 三主线叙事系统：江湖浪客/朝廷鹰犬/名门正派，各含独立剧情与年度Boss
- 主线双选机制：顺应（获得奖励但增加威胁值）或抗争（战斗/付钱换取威胁不增）
- 威胁值系统：影响年末Boss强度，目标面板显示威胁度层级
- 模块化数据、系统、UI结构
- 年月、行动力、金钱和属性成长
- 每月固定三次随机事件（主线/成长/风险三槽）
- 前两次事件选择后补充新事件
- 属性修炼和招式修炼
- 传武堂购买秘籍
- 内功心法获取与装备
- 突破奖励三选一
- 商人和战斗道具
- 行动条战斗（速度倍率可调 1x/2x/3x）
- 玩家主动选择技能或道具
- 逃跑选项（50%成功率，Boss战不可逃跑）
- 最终Boss自动触发（受威胁值加成）
- 通关/失败结算
- 局外成长与宝物解锁
- 本地存档/继续游戏
- 调试面板

## 当前定位

这是用于验证核心玩法的纵切原型。正式开发时应继续拆分数据、UI、战斗和效果系统，并使用原创美术与文案。
