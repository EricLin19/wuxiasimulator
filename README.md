# 小小侠客网页版原型

## 运行方式

直接打开 `index.html` 即可游玩。

如浏览器限制本地脚本，可在此目录启动静态服务器：

```powershell
python -m http.server 8877 --bind 127.0.0.1
```

然后访问：

```text
http://127.0.0.1:8877
```

## 已实现内容

- 三名角色与六件开局宝物
- 模块化数据、系统、UI结构
- 年月、行动力、金钱和属性成长
- 每月固定三次随机事件
- 前两次事件选择后补充新事件
- 属性修炼、计略修炼和招式修炼
- 传武堂购买秘籍
- 计略获取与融合
- 突破奖励三选一
- 商人和战斗道具
- 行动条战斗
- 玩家主动选择技能或道具
- 自动战斗
- 最终Boss自动触发
- 通关/失败结算
- 局外成长与宝物解锁
- 本地存档/继续游戏
- 调试面板

## 当前定位

这是用于验证核心玩法的纵切原型。正式开发时应继续拆分数据、UI、战斗和效果系统，并使用原创美术与文案。
