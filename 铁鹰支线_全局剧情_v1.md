# 铁鹰入局 · 全局48月剧情

## M1

```yaml
title: "雨夜血诏"
text: "户部主事陈伯钧死于家中，雨水从破窗灌入，冲不淡墙上血字：“边饷不入关，十万魂无棺。”你奉北镇抚司之命封锁现场，却发现东厂的人已经先到一步，香灰未冷，案卷却少了一页。"
```

## M2

```yaml
title: "诏狱提人"
text: "陈家遗孤被押入北镇抚司诏狱，供词上写着私通边军、畏罪牵连。孩子满脸是血，却只反复说一句：“账本在钟里。”狱卒催你画押定案，你看着那双发抖的手，忽然想起自己第一次穿上飞鱼服时，也曾信过案卷上的字。"
boss: "诏狱锁吏·常百川。北镇抚司老锁吏，铁尺、锁链、铜钥齐用，擅缴械、封招与呼唤狱卒护身。"
```

## M4

```yaml
title: "东厂来客"

boss: "东厂档头·陆怀霜。短刃快攻、雨幕影步、高闪避试刀，低血量进入“影入雨帘”短暂提升闪避与速度。"
```

boss: "西厂白灯队·素灯七番。七名白灯番子合围义庄，为首者银线控场，擅减速、缠缚与低血量召番护卫。"
