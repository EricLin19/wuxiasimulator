## 八、局外成长（v5.6 新增）

每通关/失败一局，获得若干**局外属性点**（`metaPoints`），可在主菜单「局外成长」面板中分配，影响**下一局**的开局状态。

### 8.1 可分配属性

| 属性 | 每点效果 | 说明 |
|:----:|:----:|------|
| 血量 | 新局血量 +90 | 等价于 1 次行动点修炼（血+90） |
| 内力 | 新局内力 +30 | 等价于 1 次行动点修炼（内+30） |
| 攻击 | 新局攻击 +3 | 等价于 1 次行动点修炼（攻+3） |
| 防御 | 新局防御 +3 | 等价于 1 次行动点修炼（防+3） |
| 连击 | 新局连击 +2 | 每点提升 2 连击（连击=额外攻击次数） |
| 命中 | 新局命中 +3 | 实际命中率 = 100 + 命中 - 敌方闪避 |
| 闪避 | 新局闪避 +1 | 降低被命中概率 |
| 暴击 | 新局暴击 +2 | 暴击时伤害 ×1.5 |
| 出手速度 | 新局出手速度 +0.04 | 影响行动顺序，越高越先手 |
| **金钱** | **新局额外 +100 金钱** | 基础 300，每点 +100 |

> 金钱分配效果示例：分配 3 点金钱 → 新局初始金钱 = 300 + 3×100 = **600 金**。

### 8.2 分配规则

- 每次分配消耗 1 点局外属性点（`metaPoints--`）
- 已分配点数在**下一局开局**时生效（通过 `applyMetaAllocations` 写入 `run.stats`，金钱直接加成 `run.money`）
- 分配可逆：同一次游戏中可重新分配（重新开局时生效）
- UI 入口：主菜单 → 角色选择界面 → 点击「局外成长」按钮

### 8.3 代码对应

| 代码位置 | 关键逻辑 |
|------|------|
| `content.js` L1120 | `allocations: { hp:0, qi:0, ..., money:0 }` 初始值 |
| `runSystem.js` L1425 | `applyMetaAllocations(stats, allocations)` 应用属性加成 |
| `runSystem.js` L26-34 | `moneyBonus = allocations.money × 100`，写入 `run.money` |
| `main.js` L393 | `allocateMeta(key)` 处理分配（含 `money` 键） |
| `render.js` L249 | `renderMetaModal` 渲染面板（含金钱行） |
