# 孤云逐浪 · 武林商人与秘籍装备 v5.18

> 适用于孤云逐浪主线的武林商人货品池。精选江湖气息浓厚的武学和装备，剔除名门正派/官场/邪教相关的内容。
>
> **定价基准**：`金钱 : 血量 : 内力 : 攻击/防御 = 100 : 50 : 10 : 1`
> 即 100 钱 = 50 血量 = 10 内力 = 1 攻击/防御。装备类按此基准折算后乘以装备折扣（蓝 0.5、橙 0.6、红 0.7），再根据特殊效果微调。

## 一、筛选原则

| 保留 | 剔除 |
|------|------|
| 散人/江湖/绿林气质 | 名门正派/官场/鬼教气质 |
| 草莽实用主义 | 过于精致华丽的 |
| 粗糙但有力量感 | 仙侠/玄幻色彩过重的 |
| 能跟主线产生关联的 | 与孤云逐浪主题不符的 |

## 二、武林商人布局

> **统一刷新机制**：池子全部等概率，不再区分年份品质。每次生成16个栏位：6 + 2 + 3 + 5 = 16。

```
┌─────────────────────────────────────────┐
│          武林商人           [刷新 N] [离开]│
├──────────────────┬──────────────────────┤
│ 外功秘籍 ×6       │ 装备(武器/防具) ×3     │
│ [卡][卡][卡]      │ [卡][卡][卡]          │
│ [卡][卡][卡]      │                      │
├──────────────────┼──────────────────────┤
│ 内功秘籍 ×2       │ 丹药 ×5              │
│ [卡][卡]          │ [卡][卡][卡][卡][卡]   │
└──────────────────┴──────────────────────┘
```

### 刷新规则

| 项目 | 说明 |
|------|------|
| **基础刷新次数** | 1 次（初始即获得） |
| **散人决心加成** | 每 1 点散人决心 +1 次刷新机会。动态计算：决心增加时刷新次数立即跟随增加 |
| **总刷新次数** | `1 + 散人决心 - 已用次数` |
| **刷新范围** | 全部 16 个栏位（外功秘籍 + 内功秘籍 + 装备 + 丹药）全部刷新 |
| **池子概率** | 不再按年份区分品质，池中所有货品**等概率**随机抽取 |

## 七、实现参考

### 7.1 进货生成

```js
// generateWandererMerchantStock：生成16个栏位
function generateWandererMerchantStock(run) {
  const pool = DATA.wandererMerchantPool;

  // 工具函数：等概率无放回抽取 n 个
  const pick = (arr, n, filter) => {
    let avail = filter ? [...arr].filter(filter) : [...arr];
    const result = [];
    for (let i = 0; i < n && avail.length > 0; i++) {
      const idx = Math.floor(Math.random() * avail.length);
      result.push(avail[idx]);
      avail.splice(idx, 1);
    }
    return result;
  };

  // 外功秘籍 6 本（排除已学会的）
  const manuals = pick(pool.manuals, 6,
    m => !run.skills.includes(m.id) && !run.trainingSkills.includes(m.id)
  ).map(m => ({ kind: "manual", id: m.id, price: m.price }));

  // 内功秘籍 2 本（排除已拥有的）
  const arts = pick(pool.internalArts, 2,
    a => !run.internalArts.includes(a.id)
  ).map(a => ({ kind: "internalArt", id: a.id, price: a.price }));

  // 装备 3 件（武器 + 防具合并池，等概率）
  const allEquip = [
    ...pool.weapons.map(w => ({ kind: "weapon", ...w })),
    ...pool.armors.map(a => ({ kind: "armor", ...a }))
  ];
  const equipment = pick(allEquip, 3).map(e =>
    ({ kind: e.kind, id: e.id, price: e.price }));

  // 丹药 5 种（等概率）
  const pills = pick([...pool.pills], 5).map(p =>
    ({ kind: "item", id: p.id }));

  run.merchantStock = [...manuals, ...arts, ...equipment, ...pills];
}
```

### 7.2 刷新机制

```js
// 初始化（首次进货/定期进货时调用）
// 追踪已用刷新次数，可用 = (1 + 散人决心) - 已用
// 散人决心增加时，可用刷新自动 +1
function initWandererMerchant(run) {
  generateWandererMerchantStock(run);
  // 向后兼容旧存档 _merchantRefreshes → _merchantRefreshesUsed
  if (run._merchantRefreshesUsed === undefined && run._merchantRefreshes !== undefined) {
    const maxRef = 1 + (run.wandererResolve || 0);
    run._merchantRefreshesUsed = Math.max(0, maxRef - run._merchantRefreshes);
    delete run._merchantRefreshes;
  } else {
    run._merchantRefreshesUsed = run._merchantRefreshesUsed || 0;
  }
}

// 手动刷新（点击刷新按钮时调用）
function refreshWandererMerchant(run) {
  const maxRefreshes = 1 + (run.wandererResolve || 0);
  const used = run._merchantRefreshesUsed || 0;
  if (used >= maxRefreshes) return false;
  run._merchantRefreshesUsed = used + 1;
  generateWandererMerchantStock(run);
  saveRun(run);
  return true;
}
```
