# 孤云逐浪 · 奇遇系统设计 v5.4

> 适用于孤云逐浪主线的成长/风险事件池。主线事件已移入 36 月叙事架构，本文件仅含每月随机刷新的非主线奇遇。

## 一、分布设计

| 类型 | 占比 | 数量(20池) | 说明 |
|------|:----:|:----------:|------|

## 二、传功类（10%，2个）

### 1. 龙井谷铁衣遗志

**来源**：韩铁衣生前最后一场战斗的遗训。

> 龙井谷废墟的石壁上刻着韩铁衣最后的手迹。不是招式，而是一个命令：「散人不打花拳——打一刀要让对面流血三天；扛不住就练筋骨，练到扛住为止。」你用手掌抚过石壁上的刻痕，每一条都入石三分。这是他死前用最后的力气刻上去的。不是留给你学的——是留给你活的。

| 属性 | 内容 |
|------|------|
| 事件ID | `wanderer_heritage_tieyi` |
| 触发条件 | M10之后，已完成龙井谷血战 |
| 效果 | 获得特性「铁衣遗志」：流血招式层数+3，防御永久+10 |

### 2. 灰袍手札

**来源**：孟天衡暗中传递的武学密要。

> 信鸽落在窗台上，腿上绑着一卷薄纸。展开来看是几页手写的修行心得——笔迹你认得，和那份安置营地图一模一样。孟天衡在末尾写了一行小字：「这些东西武盟不让外传。但你用得着。」没有署名，没有问候。他把命押给了你，连一句废话都不多说。

| 属性 | 内容 |
|------|------|
| 事件ID | `wanderer_heritage_meng` |
| 触发条件 | M16之后，已收到孟天衡的信 |
| 效果 | 内力上限+80，获得特性「明心」：负面状态持续时间-20% |

## 三、道具类（10%，2个）

### 3. 外编队遗物

**来源**：废弃矿坑中散人留下的遗物。

> 在一处废弃的外编队矿坑中，你发现了散人矿工藏起来的包裹。里面有几颗丹药、还有一封没寄出去的家书。家书最后一段写着「娘，矿坑塌了一次，我腿断了。不过没事，有个姓韩的大哥偷偷塞给我药。」日期是老李死前三个月。

| 属性 | 内容 |
|------|------|
| 事件ID | `wanderer_item_relic` |

以下事件均与主线核心人物相关，每月随机刷出，奖励随年份递增。

### 5. 铁衣遗训·锻体

> 深夜在龙井谷废墟独自练功。韩铁衣说过「散人没有师门，身体就是最后的武器。」你一拳一拳打在焦黑的树干上，直到月光偏西。

### 6. 龙井废墟·吐纳

> 龙井谷废墟的清晨有股特别的清气。韩铁衣生前每天早上都在这里打坐——他说「谷里的风会教你呼吸」。你盘腿坐在他坐过的石头上，让风吹进丹田。

| 事件ID | 属性 | 基础值 | Y1 | Y2 | Y3 |

## 五、切磋打斗类（30%，6个）

敌人类型与主线势力相关，难度随年份梯度递增。

| 2 | 2~3 | 赏金猎人、突击队、小Boss级 |
| 3 | 3~4 | 精英守卫、护法亲卫 |

### 详细设计

#### 9. 武盟残部

> 几个赵崇岳死后的残余堂口弟子在乡间横行。他们没有头目了，但更加危险——因为无所顾忌。领头的一个看见了你的剑，咧嘴一笑：「这把剑我见过……你就是杀了赵堂主那个散人。」他拔刀的时候手在发抖，但眼神是饿狼的眼神。

| 战斗 | 赵崇岳残部堂口弟子×2~3，小概率带一个执事 |

#### 10. 赏金猎人

> 五千两的悬赏让这些人从四面八方赶来。装备精良，配合默契，每一个都号称「猎过不下十个散人」。领头的独眼汉子打量着你：「真人比画像精神。不过无所谓——脑袋砍下来都一样值钱。」

| 战斗 | 赏金猎人团×3~4，配合默契，有合击技能 |

#### 11. 叛徒清剿

> 散人队伍里又少了一个人。追了两天在一间客栈堵住了他——正在跟武盟的人做交易。他看见你的时候脸上的表情不是愧疚，而是如释重负：「别怪我。我家还有老娘。」

| 战斗 | 叛徒+接应的武盟弟子×2 |
| 分支 | 战胜后可选择「放了」（获取情报）或「处决」（震慑其他叛徒） |

#### 12. 擂台立威

> 「散人现在不好惹了，你是领头的——敢不敢让我试试？」一个江湖刀客在散人营地外面摆下擂台。围观的人越来越多，你发现其中混着几张陌生的脸——也许是想投奔的散人，也许是武盟的探子。无论如何，这一场必须赢。

| 战斗 | 流浪刀客/拳师/剑客×1（Rank随年份） |

#### 13. 护送家眷

> 三个散人的家眷要穿过武盟控制区转移到安全地带。都是老人和孩子。路上走不快，但夜色是最好的掩护。远处火把的光芒忽明忽暗——巡逻队正在靠近。

| 战斗 | 巡逻追兵×3~4；失败则失去护送对象（影响散人营地士气） |

#### 14. 安置营看守

> 孟天衡的地图上标注了这个安置营——里面关着三十几个散人。看守换班的间隙只有一炷香的时间。你和几个兄弟翻过围墙时，看守长正在打牌。牌桌上的银子大概也是从散人身上搜刮来的。

| 战斗 | 看守长+守卫×4（小Boss级，Rank 3） |
| 胜利 | 救出散人→散人营地人口增加，士气提升 |

## 六、纯金钱类（30%，6个）

与孤云逐浪主线剧情松散关联的散人日常营生。所有金钱奖励按年份递增。

| # | 事件ID | 名称 | 来源 | 年份 | 金钱(Y1/Y2/Y3) |
|---|------|------|------|:----:|:----:|
| 15 | `wanderer_coin_escort` | 替散人运镖 | 韩铁衣生前安排的物资线 | 1-3 | 150 / 210 / 300 |
| 16 | `wanderer_coin_rebuild` | 龙井谷重建 | 帮散人重建营地干体力活 | 1-3 | 120 / 180 / 260 |
| 17 | `wanderer_coin_intel` | 孟天衡情报费 | 替孟天衡传递情报获取酬劳 | 2-3 | — / 240 / 360 |
| 18 | `wanderer_coin_fangping` | 方平的私活 | 跟方平一起接江湖零活挣快钱 | 1-3 | 180 / 250 / 350 |
| 19 | `wanderer_coin_labor` | 安置营杂工 | 混入安置营做苦力顺便侦查 | 1-3 | 100 / 160 / 240 |
| 20 | `wanderer_coin_market` | 散人集市帮手 | 帮散人摆摊交易、跑腿送货 | 1-3 | 130 / 190 / 280 |

### 详细设计

#### 15. 替散人运镖

> 韩铁衣生前搭了一条散人自己的物资线——从山里收药材，运到城郊黑市换粮食。武盟封锁之后这条线断了。一个老镖头找到你：「路不好走，但货得送。沿路的散人都指着这批粮食。」他递过来一张皱巴巴的路线图——上面标注的每一个安全点，都是韩铁衣拿血换来的。

| 事件ID | `wanderer_coin_escort` |
| 触发 | Y1-3，随机 |
| 奖励 | 运气好可额外获得跑镖途中捡到的随机丹药×1 |

#### 16. 龙井谷重建

> 血战之后的龙井谷需要重建。散人里面老的小的都有，能扛能打的没几个。你卷起袖子搬石头、修棚子、挖水沟——干的都是最重的体力活。天黑收工时一个老妇人端来一碗热汤：「小伙子，我们家男人要是还活着，也是你这把年纪。」

| 事件ID | `wanderer_coin_rebuild` |
| 触发 | M10之后，Y1-3 |
| 效果 | 额外小幅提升散人营地士气 |

#### 17. 孟天衡情报费

> 一个从不露面的人托人送来一封信和十两银子。信上只有几个字：「城西茶楼，申时，留意左手戴铁指环的人。」你去看了——那人和武盟的人接头，说出了三个散人据点的位置。你把他摁在巷子里的时候，他塞过来一包银子：「兄弟别杀我，我知道更多。」你没有杀他——但把他知道的全问出来了。五天后，又有人送来一封信，这次的银子里夹了一张纸条：「干得不错。——孟」

| 事件ID | `wanderer_coin_intel` |
| 触发 | M16之后，Y2-3 |
| 限制 | 需先获得孟天衡信任 |

#### 18. 方平的私活

> 方平在村口朝你招手：「别绷着一张脸了，出来挣点钱。」他接了一单江湖活——给一个退隐的老拳师修院子。活不累，但老拳师看你们手脚麻利，多结了工钱不说还指点了几招：「年轻人，拳头是最后的东西。靠力气吃饭不丢人。」方平把银子分你一半：「下次换个来钱快的，这个老家伙太能唠了。」

| 事件ID | `wanderer_coin_fangping` |
| 触发 | M2之后，Y1-3 |
| 效果 | 有概率随机属性临时+1（老拳师指点） |

#### 19. 安置营杂工

> 混进武盟的安置营当临时杂工——搬货、扫地、倒马桶。一天累死累活只挣几个铜板，但你趁着干活把安置营的布局摸了个透：看守换班时间、仓库位置、哪面墙最矮。走的时候领班骂你偷懒，你低着头不吭声——心里已经把救人的路线画好了。

| 事件ID | `wanderer_coin_labor` |
| 触发 | Y1-3 |
| 效果 | 下次安置营相关战斗时命中+5（地形情报加成） |

#### 20. 散人集市帮手

> 散人营地外有个小集市——卖草药的、修兵器的、缝衣服的，全是散人自己。今天人手不够，你帮忙搬货、跑腿送东西、替一个不识字的大娘读账本。一天下来腿都跑细了，但集市的散人们凑了一包碎银子塞给你，一个老裁缝还顺手给你缝好了袖口的破洞。

| 事件ID | `wanderer_coin_market` |
| 触发 | M6之后，Y1-3 |
| 效果 | 有概率获得「散人缝补」：下次战斗中受到的第一次伤害-15% |

## 八、实现建议

```js
// 孤云逐浪奇遇池分五个子池
DATA.wandererAdventures = {
  heritage: [ /* 传功2个 */ ],
  item:      [ /* 道具2个 */ ],
  stat:      [ /* 属性4个 */ ],
  fight:     [ /* 打斗6个 */ ],
  coin:      [ /* 金钱6个 */ ]
};

// 刷新时按权重抽取: 10/10/20/30/30
function makeWandererGrowthPool(run) {
  const roll = Math.random() * 100;
  if (roll < 10) return pickFrom('heritage');
  if (roll < 20) return pickFrom('item');
  if (roll < 40) return pickFrom('stat');
  if (roll < 70) return pickFrom('fight');
  return pickFrom('coin');
}
```

## 九、金钱收益对照基准

按全局基准 `金钱:血量:内力:攻击/防御 = 100:50:10:1`：

| 年份 | 金钱事件均收益 | 折合血量 | 折合内力 | 折合攻/防 |
|:----:|:----------:|:----:|:----:|:----:|
| 1 | ~140 | 70 | 14 | 1.4 |
| 2 | ~205 | 103 | 21 | 2.1 |
| 3 | ~298 | 149 | 30 | 3.0 |

与每月平均成长（血量+90、内力+20、攻/防+2）对比，金钱事件提供约 60-75% 的等效月成长价值，符合辅助定位。
