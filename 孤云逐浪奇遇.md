## 一、刷新机制

- **每月刷新 6 个奇遇**，按概率权重从全池加权无放回抽取
- **玩家六选三**：6 个事件同时展示，剩余选择次数 3，选完即止
- 概率权重：传功 10% / 道具 10% / 加属性 20% / 切磋打斗 30% / 纯金钱 30%
- 若传功全部领取或道具未解锁，其权重并入加属性

### 权重分布表

| 类型 | 权重 | 数量 | 说明 |
|------|:----:|:----:|------|
| 纯金钱类 | 30% | 6 | 随时间推移加入奇遇（不变） |
| 切磋打斗 | 30% | 3 | 通用池三种，随年份增加难度 |
| **加属性类** | **40%** | — | 含传功、道具、加属性三个子类 |

加属性类内部细分：

| 子类 | 占全局 | 占加属性内部 | 数量 | 说明 |
|------|:----:|:---------:|:----:|------|
| 传功 | 10% | 25% | 6 | 剧情关联，每局各限领一次 |
| 道具 | 10% | 25% | 2 | 可反复触发 |
| 加属性 | 20% | 50% | 5 | 通用属性成长，无领取限制 |

**概率吸收规则**：若当月无可用传功事件（全部已领取或未解锁），其 10% 概率归入加属性。若当月无可用道具事件（游商秘货 Y2 才解锁），其 10% 概率也归入加属性。加属性最高可达 40%。

**传功领完规则**：6 个传功每局各限领一次，领完后传功出现概率归 0%。

## 二、传功类（10%，6个）

> 全部与主线剧情人物关联，每局游戏各限领一次。6 个领完后传功概率归零，其权重并入加属性。主角初始 `collectedHeritages = []`。

### 1. 铁衣遗训·血偿

**来源**：韩铁衣生前最后一场战斗的遗训。

> 龙井谷废墟的石壁上刻着韩铁衣最后的手迹。不是招式，而是一个命令：「散人不打花拳——打一刀要让对面流血三天。」你用手掌抚过石壁上的刻痕，每一条都入石三分。这是他死前用最后的力气刻上去的。

| 属性 | 内容 |
|------|------|
| 事件ID | `wanderer_heritage_tieyi` |
| 触发月 | ≥ M10（龙井谷血战后，韩铁衣已死） |
| 效果 | 获得特性「血偿」：流血招式层数+3，无论武器类型均生效 |

### 2. 灰袍手札

**来源**：孟天衡暗中传递的武学密要。

> 信鸽落在窗台上，腿上绑着一卷薄纸。展开来看是几页手写的修行心得——笔迹你认得，和那份安置营地图一模一样。孟天衡在末尾写了一行小字：「这些东西武盟不让外传。但你用得着。」没有署名，没有问候。他把命押给了你，连一句废话都不多说。

### 3. 铁衣遗训·锻体

> 深夜在龙井谷废墟独自练功。韩铁衣说过「散人没有师门，身体就是最后的武器。」你一拳一拳打在焦黑的树干上，直到月光偏西。三个月下来，拳骨磨出了茧子，树干也终于被你打断了一棵——那不是树输了，是你的身体终于追上了韩铁衣说的那个标准。

> 你在歇脚的破庙里发现一个旧的不能再旧的包袱，上面缝着几个歪歪扭扭的字——「有用的留下」。打开一看是个散人传下来的药囊，里面装着几颗丹药。不知道经过多少人的手，也不知道下一个传给谁。你拿了两颗，把剩下的重新包好放回原处。江湖就是这样——今天你拿，明天你放。

### 详细文案

#### 9. 破绽寻击

> 对着木桩反复打同一个点。一开始是拳头大的范围，几个月后缩小到铜钱大小，最后连硬币大的疤都能精准命中。散人的招式不花哨——能打死人的就是好招。

#### 10. 连环手法

> 苦练连招衔接。第一拳没收回来第二拳已经出去了，第三拳不用看就知道打在哪里。快到不需要想——不是招式快，是节奏快，是你没时间犹豫。等你犹豫的时候对手已经还手了。

#### 11. 飘萍步

> 踩在溪水里的石头上练步法。一开始每步都滑，摔得浑身湿透。练到后来脚下像长了眼睛——不靠力气，不靠速度，靠的是对重心的感觉。身如飘萍，随波逐流，却始终不翻。

#### 12. 疾影式

> 对着铜镜练出招。一开始看得见自己的手。练到后来镜子里只剩残影。出手快不是手快——是脑子快。脑子比你手快的那一瞬间，你已经赢了。

#### 13. 鹰目诀

> 山谷里追着飞鸟的轨迹练眼力。从看不清到能数出翅膀上的羽毛。你发现所有的鸟起飞之前翅膀都会先收一下——就像所有对手出招前，肩膀都会先动一下。看见了，就躲得开。

### 详细设计

#### 14. 林中伏击

> 穿过密林时听到前方树枝折断的声响——不是野兽，是人。你还没来得及退，身后的退路也被人堵住了。领头的喽啰提着刀从树后走出来：「散人？正好——拿你回去交差。」

| 战斗 | 林中伏击 ×2~3（年份变体），胜利获得额外金钱 |

#### 15. 悬赏缉拿

> 城门口贴着你的画像，下面的赏金又涨了。几个专门做悬赏生意的狠角色盯上了你——其中一个手里还拿着你上次战斗时掉落的剑穗：「这个是你的吧？找了你三天了。」

| 战斗 | 悬赏缉拿 ×1~2（年份变体），胜利获得额外经验 |

#### 16. 擂台切磋

> 「散人现在不好惹了——敢不敢让我试试？」一个江湖人当街摆下擂台。围观的人越来越多，其中有几张陌生的脸——也许是想投奔的散人，也许是武盟的探子。无论如何，这一场必须赢。

| 战斗 | 擂台切磋 ×1（年份变体），胜利获得额外金钱与经验 |

## 十、实现参考

> 每月生成 6 个奇遇，玩家六选三。`weightedPickMultiple` 按权重无放回抽取，保证同一事件不会重复出现。

```js
// 孤云逐浪奇遇池分五个子池，每个事件带 unlockMonth 和 yearly data
DATA.wandererGrowthEvents = {
  heritage: [
    // 6 个传功事件，每局各限领一次
    { id:"wanderer_heritage_tieyi",  unlockMonth:10 },
    { id:"wanderer_heritage_meng",   unlockMonth:16 },
    { id:"wanderer_heritage_hp",     unlockMonth:10 },
    { id:"wanderer_heritage_qi",     unlockMonth:10 },
    { id:"wanderer_heritage_break",  unlockMonth:10 },
    { id:"wanderer_heritage_step",   unlockMonth:16 }
  ],
  item: [
    { id:"wanderer_item_relic",      unlockMonth:1 },
    { id:"wanderer_item_merchant",   unlockYear:2 }
  ],
  stat: [
    { id:"wanderer_stat_crit",  statKey:"crit",  values:{y1:2,y2:3,y3:5},   unlockMonth:1 },
    { id:"wanderer_stat_combo", statKey:"combo", values:{y1:2,y2:3,y3:5},   unlockMonth:1 },
    { id:"wanderer_stat_dodge", statKey:"dodge", values:{y1:2,y2:3,y3:4},   unlockMonth:1 },
    { id:"wanderer_stat_speed", statKey:"speed", values:{y1:0.04,y2:0.06,y3:0.10}, unlockMonth:1 },
    { id:"wanderer_stat_hit",   statKey:"hit",   values:{y1:3,y2:5,y3:7},   unlockMonth:1 }
  ],
  fight: [
    { id:"wanderer_fight_ambush",  pool:"ambush",  count:[2,3,3], unlockMonth:1 },
    { id:"wanderer_fight_bandit",  pool:"bandit",  count:[1,2,2], unlockMonth:1 },
    { id:"wanderer_fight_fighter", pool:"fighter", count:[1,1,1], unlockMonth:1 }
  ],
  coin: [
    // 6 个金钱事件，与原来相同
  ]
};

// 概率权重（全局）
const WEIGHTS = { heritage:10, item:10, stat:20, fight:30, coin:30 };

// applyEventWeights: 应用权重调整（传功/道具不可用时并入加属性）
function applyEventWeights(events, heritageAvail, itemAvail) {
  const _ha = (heritageAvail !== undefined) ? heritageAvail : 99;
  const _ia = (itemAvail !== undefined) ? itemAvail : 99;
  const adjusted = events.map(e => {
    let w = e.weight || 10;
    if (e.category === "高手传功" && _ha === 0) w = 0;
    if (e.category === "道具" && _ia === 0) w = 0;
    return { ...e, _w: w };
  }).filter(e => e._w > 0);
  // 不可用类别权重并入 stat
  const lostHeritage = (_ha === 0) ? 10 : 0;
  const lostItem = (_ia === 0) ? 10 : 0;
  const totalLost = lostHeritage + lostItem;
  if (totalLost > 0) {
    const statEvents = adjusted.filter(e => e.category === "属性");
    if (statEvents.length) {
      const bonusPerStat = totalLost / statEvents.length;
      statEvents.forEach(e => { e._w += bonusPerStat; });
    }
  }
  return adjusted;
}

// weightedPickMultiple: 加权无放回抽取 n 个事件
function weightedPickMultiple(events, n) {
  if (events.length <= n) return [...events];
  const result = [];
  const remaining = events.map(e => ({ ...e }));
  for (let i = 0; i < n && remaining.length > 0; i++) {
    const totalW = remaining.reduce((s, e) => s + (e._w || 10), 0);
    if (totalW <= 0) { result.push(remaining.shift()); continue; }
    const roll = Math.random() * totalW;
    let acc = 0, pickIdx = remaining.length - 1;
    for (let j = 0; j < remaining.length; j++) {
      acc += remaining[j]._w || 10;
      if (roll < acc) { pickIdx = j; break; }
    }
    result.push(remaining[pickIdx]);
    remaining.splice(pickIdx, 1);
  }
  return result;
}

// refreshWandererEvents: 每月生成6个奇遇
function refreshWandererEvents(run) {
  const growthEvents = makeWandererGrowthPool(run);  // 返回 applyEventWeights 后的全量
  const riskEvents = makeWandererRiskPool(run);       // 同上
  const allEvents = [...growthEvents, ...riskEvents];
  run.events = weightedPickMultiple(allEvents, 6);
  run.eventRemaining = 3;  // 六选三
}
```
